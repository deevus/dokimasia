# Contributing

## Contribution trust policy

This project uses a vouch-based contribution model for pull requests.

Pull requests from unknown or unvouched contributors may be automatically closed. If you want to contribute, please open an issue first and discuss the change with a maintainer before submitting implementation work. Maintainers may vouch for contributors at their discretion.

We do not accept unsolicited drive-by pull requests generated primarily by automated coding agents.
